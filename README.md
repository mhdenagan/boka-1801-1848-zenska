# Женска имена у Протоколима крштених Храма Св. Оца Николаја у Боки (1801–1848)

author: Димитрија Рашљић

affiliation: Универзитет у Београду, Филозофски факултет

email: dimitrija.rasljic@f.bg.ac.rs

orcid-id: 0000-0002-3407-5831

date: "23.04.2026"

Рад представља попис и анализу женских световних имена која су бележена од 1801. до. 1848. године у Протоколима крштених Храма Св. Оца Николаја у Боки у Банату. Имена крштене деце су пописана у потпуности, са датумом рођења, датумом крштења, именом мајке и именом кума (уколико је кум женског рода)
